#  Porto Hotels & Resorts - End-to-End QA Project

##  Project Overview
A comprehensive Quality Assurance project for the **Porto Hotels & Resorts** platform. This repository demonstrates a full-cycle testing approach, combining robust **Automation Testing** with detailed **Manual Testing** to ensure a flawless user experience, with a special focus on the critical **Room Booking Flow**.

##  Tech Stack & Frameworks
- **Language:** Java
- **Automation:** Selenium WebDriver
- **Test Runner & BDD:** TestNG, Cucumber (Gherkin)
- **Architecture:** Page Object Model (POM)
- **Build Tool:** Maven
- **Manual Testing:** Excel (Test Cases, Bug Reports, Execution Dashboard)
- **Reporting:** Extent Reports / Allure Reports

##  Key Highlights
 **BDD Approach:** Writing readable and maintainable test scenarios using Cucumber.
 **POM Design Pattern:** Ensuring clean code, reusability, and easy maintenance.
 **Comprehensive Coverage:** Testing Homepage, Navigation, Booking Search, Accommodation, and Contact forms.
 **Manual Testing Artifacts:** Included professional Test Cases and Bug Reports with Severity/Priority classifications.
